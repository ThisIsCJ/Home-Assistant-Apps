# Changelog

## 1.0.0

- Initial release
- Visual drag-and-drop designer for openHASP panels
- Support for all common openHASP object types
- Multi-page support (pages 0-11)
- Property editor with type-specific controls
- Export to openHASP JSONL format
- Import existing JSONL designs
- Save/load designs within the add-on
- Configurable panel dimensions
- Snap-to-grid positioning (5px)
- Keyboard shortcuts (Delete, Ctrl+D, Escape)
